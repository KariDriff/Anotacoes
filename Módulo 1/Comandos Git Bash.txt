git init          | Inicializa o Git na pasta atual e possibilita o git a gerenciar o código.
git config        | Faz uma configuração do Git, que pode ser de forma --global ou de forma --local. Use --list para ver a lista completa das configurações. Use --global --unset [config] para remover uma configuração e, depois, --global [config] para adicioná-la com outra configuração.
git add           | Adiciona um arquivo ou pasta ao commit. Use * para dicionar tudo que foi modificado.
git commit -m     | Cria ou edita um commit com uma mensagem.
git status        | Checa o status dos arquivos, se estão Untracked, Unmodified, Modified ou Staged.
git rm            | Remove um arquivo ou pasta do commit.
git remote        | Trabalha com o repositório remoto. Use add para adicionar o repositório local a um repositório remoto. Use -v para ver os repositórios remotos cadastrados.
git push          | "Empurra" o código do repositório local para o repositório remoto.
git pull          | "Puxa" o código do repositório remoto para o repositório local.

clear             | Limpa o Git Bash.
exit              | Fecha o Git Bash.

mkdir             | "MaKe DIRectory", Cria uma pasta.
cd                | "Change Directory", Muda para a pasta especificada.
cd ..             | Volta para a pasta anterior.
ls                | "LiSt", Lista todos os arquivos e pastas da pasta atual.
ls -a             | Lista todos os arquivos e pastas da pasta atual incluindo pastas ocultas.
pwd               | Mostra o caminho completo da pasta atual.
———————————————————
cat               | Lê uma chave .pub.
mv [fl/dr] ./[dr] | "MoVe", Move um arquivo ou pasta da pasta atual para outra pasta.
mv [fl dir] [dir] | Move um arquivo ou pasta de outra pasta para a pasta especificada.

openssl           | Abre a SSL especificada e a aplica em um arquivo. Use sha1.
